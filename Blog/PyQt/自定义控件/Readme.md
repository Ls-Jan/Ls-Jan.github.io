
写了不少的自定义控件，自认为泛用性挺足，例如
- 跑马灯容器控件：鼠标悬浮时滚播控件内容，方向速率可定
- 弹窗式容器控件：行为类似于组合款点击时弹出的列表，但仅会点击容器外部某处才消失(点击容器内不会消失)
- 方位式容器控件：可以将控件固定在指定控件的某处(九个位置可选)
- 加载动画遮罩控件：会遮挡指定控件并显示加载动画
...
